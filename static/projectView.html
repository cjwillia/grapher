Grapher is a tool for organizing projects visually. Use the buttons on the control panel above the project to make edits to the project on the canvas


Keyboard shortcuts: [B] - 'Add Task' | [C] - 'Add Connection' | [D] - 'Delete Node' | [F] - Remove all Connectors from Node


Pressing the key of the mode you are currently in will return you to the default select mode.
